# Sparrow


CSEC 731 Python Web Server


Author: Jason Howe


## Installation

Clone the repository done </br>
`git clone https://github.com/P0nt14c/sparrow.git`


## Usage 
To start the webserver run: </br>
`python3 sparrow.py`

### Request Checking
To check a HTTP Request in a file: </br>
`python3 sparrow.py <path/to/request>`
